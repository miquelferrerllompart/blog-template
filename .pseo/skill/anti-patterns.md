# Anti-patterns

Things that must NEVER happen in pSEO work:

- Never generate freeform AI content without a schema
- Never use AI to generate titles (use templates)
- Never publish all pages at once (batch rollout)
- Never skip niche context validation
- Never use a generic template across content types (each type needs its own component)
- Never mix content (JSON) and presentation (templates) in the same layer
