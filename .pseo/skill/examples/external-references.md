# External References — Examples

## JSON format

```json
{
  "title": "Descriptive Article Title",
  "url": "https://...",
  "source": "Publisher Name"
}
```

## Full array example

```json
"external_references": [
  {
    "title": "The Value of Getting Personalization Right",
    "url": "https://www.mckinsey.com/capabilities/growth-marketing-and-sales/our-insights/the-value-of-getting-personalization-right-or-wrong-is-multiplying",
    "source": "McKinsey"
  },
  {
    "title": "How to Build Sustainable Growth",
    "url": "https://hbr.org/2023/01/sustainable-growth-strategies",
    "source": "Harvard Business Review"
  },
  {
    "title": "The Ultimate Guide to Customer Engagement",
    "url": "https://blog.hubspot.com/service/customer-engagement",
    "source": "HubSpot"
  }
]
```
