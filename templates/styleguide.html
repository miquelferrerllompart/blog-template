Icon Library

Custom SVG pictograms using <symbol> + <use href="#id">. Included via {{">icons"}} partial. Themed with 3 CSS custom properties that cascade into the sprite.

--ic-1: primary stroke/fill (default: var(--accent))
--ic-2: background fill (default: var(--accent-light))
--ic-3: secondary detail (default: var(--accent-dark))
